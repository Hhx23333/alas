## 小开特供版Alas

特性：

重启时，解决GG弹窗。

可在Alas内配置的倍率小开。

一个简单的仪表盘。

出现未知界面，直接重启。

敏感项目自动重启并以正常数据攻打。

敏感项目，前排战力大于16500（可配置）强制重启。

紧急委托前排可以选择更换为10级以上的驱逐舰。

可以选择在一键退役时尝试不保留没满星的舰船（危险选项！操作前阅读说明）
